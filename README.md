# Projekt pro pojištění #
- je vytvořený jako závěrečná práce pro absolování rekvalifikačního kurzu na tvorbu www aplikací v javascriptu v síti poskytovatele vzdělávacích kurzů pro IT ITnetwork.cz
- obsahuje CRUD operace pro pojištění a pojištěnce, které lze ukládat do databáze MySQL
- Technologický stack: JavaScript,Html,CSS,Bootstrap,MySQL
